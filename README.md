# Develop

## Development server

Run `python manage.py runserver` for a dev server. Navigate to `http://localhost:8000/`. The application will automatically reload if you change any of the source files.

## Code scaffolding

Run `python manage.py startapp app_name digital_book_backend_django/apps/app_name` to generate a new app. <br> 
For more info see [startapp](https://docs.djangoproject.com/en/5.1/ref/django-admin/#startapp).
